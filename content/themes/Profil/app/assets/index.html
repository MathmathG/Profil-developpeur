Hello, je suis Mathieu développeur web

Aprés avoir passé une formation de développeur web et web mobile avec l'organisme O'clock je suis actuelement à la recherche d'une entreprise pour un contrat de professionnalisation de concepteur devloppeur web.

En savoir plus Contact
J'ai toujours été attiré par le milieu de l'informatique et lors de mes divers recherches sur le web, je suis arrivé sur le site d'O'clock. J'ai donc commencé cette formation. Je me suis familiarisé avec différends languages comme HTML et CSS, Javascript Jquery, PHP, et lors de ma spécialisation j'ai choisi WordPress car pour moi c'étais le meilleur moyen d'allier le back et le front. A l'avenir j'aimerai aussi me spécialiser sur la bibliothèque Javascript React. Cette formation m'a donc permis d'acquérir divers compétences mais avant tout une passion.
